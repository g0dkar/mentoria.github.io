# Um Programa para Programar

Nosso último passo antes de escrever nosso primeiro programa é instalar um programa que nos ajuda a criar programas!

Esses programas são normalmente chamados de IDE: Integrated Development Environment, em português: Ambiente Integrado para Desenvolvimento.
